    • Les balises HTML fréquemment utiliser sont :

    • La balise <meta description>
    • La balise <ul>, <li > et <ol> (liste)
    • La balise <br> (saut de ligne)
    • La balise <title>
    • Les balises titre <h1>, <h2>, <h3>, <h4>, <h5>, <h6>
    • La balise <strong> (mise en gras d'une partie de texte)
    • La balise <p> (paragraphe)

      • À savoir

    • Le HTML est un langage informatique qui permet de créer un site internet. Tous les sites existants fonctionnent à travers ce langage. Un site internet est une traduction de ce langage. Il est donc indispensable pour la création d'un site internet ainsi que pour une bonne conception.

    Css nous sert a apporter du disgne a notre html.

    Il va nous permettre d'ajoutez:

    • La couleur
    • La taille
    • La disposition
    • Du style
    • la creation
    • La balise <strong> (mise en gras d'une partie de texte)

    Il y a trois façon de définir des styles CSS :

    1. Dans le corps du code HTML.
    2. Dans Html avec un tag style.
    3. Dans une feuille de style totalement séparée du code HTML.

    À quoi sert Bootstrap.

  • Bootstrap est une collection d'outils utiles à la création du design de sites et d'applications web. C'est un ensemble qui contient des codes HTML et CSS, des formulaires, boutons, outils de navigation et autres éléments interactifs, ainsi que des extensions JavaScript en option.

    • À savoir

  • Le site que vous consultez actuellement est Responsive, ce qui veut dire que quelle que soit la taille de l'écran mon site s'adaptera.

  • Si vous changez la taille de l'écran alors ce site s'adaptera à votre écran. Car selon la taille d'écran ce ne sera pas le même affichage pour des raisons de logique
