CSS

Quel est le rôle CSS ?

CSS sert à définir un ensemble de règles stylistiques à l'intérieur de pages web codées en HTML ou en XML. C'est l'un des outils principaux du développement web permettant de créer la structure informatique d'une page internet.

Pourquoi Css est indispensable ?

Il est indispensable car il apporte le style. C'est a dire que Css permet de donner une taille de centrer des élement de donner de la couleur ect. Voila pourquoi Css est indispensable.

CSS se lie avec HTML ?

Oui. Car CSS nous permet de donner du style à notre page HTML, donc sans notre squelette(HTML) le language Css ne sert à rien. Chaque langage a des fonctions précises.

Comment lié CSS et HTML ?

Dans le tag head de notre page html, utiliser le tag link suivis du nom de votre document Css dans le href comme ci-dessous.
