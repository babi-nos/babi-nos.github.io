Qu'est-ce Bootstrap ?

  • Bootstrap est un framework de développement frontal gratuit et open source pour la création de sites Web et d'applications Web .

    • C'est quoi un framework ?

  • Un Framework est une boite à outils pour un développeur web. Frame signifie cadre et work se traduit par travail. Un Framework contient des composants autonomes qui permettent de faciliter le développement d'un site web ou d'une application.

    • Quel est le principal avantage de Bootstrap ?

  • L'un des plus grands avantages de l'utilisation de Bootstrap est la compatibilité entre les navigateurs . Avec Bootstrap à vos côtés, poussez un soupir de soulagement lorsqu'il s'agit d'afficher votre page de destination sur plusieurs navigateurs. Sa compatibilité avec Google Chrome, Firefox, Safari et Internet Explorer le rend plus polyvalent.
